# 🔵 RITS Demo App — Expo Go Workshop Example

## What is this?
This is the demo app shown at the beginning of the Vibe Coding Workshop. It's a mobile app promoting RITS (Richmond Institute of Technology and Science) built with React Native and Expo.

**Workshop students:** You're looking at this as an EXAMPLE. After running this app, go back to the parent folder and open your own project folder!

## Run it:
```bash
cd rits-example
npm install
npx expo start --tunnel
```

Then scan the QR code with Expo Go on your phone.

## After the demo — go back!
```bash
cd ..
cd my-expo-app
npm install
npx expo start --tunnel
```

## Claude Prompt Inspiration

Use this as a starting point when writing your own prompt for Claude:

```
Create a React Native screen component for an Expo Go app.
I want this screen to be: [YOUR IDEA HERE]

Requirements:
- Use React Native core components only (View, Text, TouchableOpacity, ScrollView, FlatList)
- Use StyleSheet.create() for all styles
- Dark background color: #0A0A0F
- Accent color: #1E6FFF (blue) or #FFFFFF (white)
- The component must be a default export
- No external libraries beyond react-native and expo basics
- Make it look polished and fun!

Ideas for screens:
- A personal about-me page
- A quiz app
- A music player UI
- A todo list
- A restaurant menu
- A sports scoreboard
- A recipe book
```
